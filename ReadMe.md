# LU Card Reader

## GPIO Pinout

- **GPIO 2:** Keypad buttons [IN]
- **GPIO 3:** House Lights
- **GPIO 5:** Buzzer[OUT]
- **GPIO 6:** Card Reader status light
- **GPIO 8:** RFID Reset
- **GPIO 9:** Top/Roof Lights
- **GPIO 10:** RFID SS
- **GPIO 11:** RFID MOSI
- **GPIO 12:** RFID MISO
- **GPIO 13:** RFID SCK
